## 三、数据局限性说明

### 3.1 时间范围限制

- **目标时间**：2025年10月1日 - 2025年12月31日
- **实际情况**：部分内容可能涉及政策前瞻性分析（2026年），部分历史内容具有延续参考价值
- **影响**：部分记录的实际发布时间可能存在细微偏差

### 3.2 平台覆盖限制

- **未覆盖平台**：
  - 微博：受限于搜索接口，未获取有效数据
  - 抖音/视频号：短视频内容难以直接转录
  - 豆瓣小组：移民讨论组内容较为分散
  - 专业论坛（篱笆网、天涯等）：内容获取受限

- **影响**：可能遗漏部分真实用户分享和讨论

### 3.3 互动数据限制

- **点赞/评论/转发**：部分数据为基于内容热度的预估，非精确数值
- **标注方式**：预估数据标注为"预估XXX+"，真实数据直接标注数字
- **影响**：互动数据分析存在一定误差

### 3.4 可信度分级说明

| 等级 | 定义 | 判断依据 | 占比 |
|------|------|----------|------|
| A级 | 官方/可验证来源 | 政府官网、官方媒体、权威机构 | 12% |
| B级 | 专业机构/媒体 | 知名移民机构、专业媒体、律师事务所 | 28% |
| C级 | 个人/推广内容 | 中介推广、个人分享、未验证信息 | 60% |

**注意**：C级内容占比最高，使用时需注意甄别。

### 3.5 语言限制

- **收集语言**：主要为中文内容
- **未覆盖**：英文、希腊语、葡萄牙语等原文内容
- **影响**：可能遗漏部分一手官方信息

### 4.2 数据收集策略

- **分批次搜索**：避免一次性搜索导致的信息遗漏
- **多关键词组合**：覆盖马耳他、希腊、葡萄牙、塞浦路斯、意大利、拉脱维亚等多个国家
- **长尾关键词**：包含"避坑"、"后悔"、"被骗"等负面关键词
- **平台细分**：针对每个平台单独搜索，提高覆盖率

## 五、数据使用方法

### 5.1 CSV数据文件

**文件名**：`european-immigration-data-2025Q4.csv`

**包含字段**：
| 字段名 | 说明 |
|--------|------|
| 平台 | 内容来源平台 |
| 标题 | 帖子标题 |
| 内容摘要 | 核心内容摘要（200字以内）|
| 发布时间 | 格式：2025-XX-XX |
| 点赞数 | 数字或"预估XXX+" |
| 评论数 | 数字或"预估XXX+" |
| 转发数 | 数字或"预估XXX+"（如可见）|
| 发布者名称 | 账号名 |
| 发布者类型 | 个人/中介/官方媒体/专业机构/移民博主 |
| 命中关键词 | 具体命中的关键词组合 |
| 内容主题 | 政策解读/经验分享/中介推广/避坑指南/竞品对比/其他 |
| 情绪倾向 | 正面/负面/中性 |
| 可信度等级 | A级（官方/可验证）/B级（专业机构）/C级（个人/推广）|
| 数据来源链接 | URL或搜索来源说明 |

### 5.2 使用建议

1. **筛选数据**：可根据"可信度等级"筛选高可信度内容
2. **情绪分析**：可参考"情绪倾向"进行舆论分析
3. **主题分类**：可根据"内容主题"进行针对性研究
4. **时间排序**：可按"发布时间"查看最新动态
