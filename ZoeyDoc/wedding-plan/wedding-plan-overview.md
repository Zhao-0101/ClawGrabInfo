## 一、婚礼概况

### 核心信息
| 项目 | 详情 |
|------|------|
| 婚礼形式 | 中式传统婚礼 + 西式仪式 |
| 预计宾客 | 500人左右（50桌） |
| 场地要求 | 宴会厅面积≥500㎡，无柱式最佳 |
| 预算范围 | 根据酒店档次，建议预留15-30万 |
| 清明节考察 | 2026年4月4日-6日（重点考察期） |

### 50桌宴会厅要求
- **面积需求**：≥500㎡（最佳800-1200㎡）
- **层高要求**：≥6米（便于布置和灯光效果）
- **柱数要求**：无柱式最佳，有柱需确认位置
- **舞台区域**：预留足够空间搭建主舞台
