### 下午：酒店A深度考察

**目标酒店建议**：泉州荣誉国际酒店 / 泉州泰禾洲际酒店 / 泉州富力万达文华酒店

| 时间 | 事项 | 检查要点 |
|-----|------|---------|
| 14:00-14:30 | 与酒店销售见面 | 确认预约，介绍需求 |
| 14:30-15:30 | **参观宴会厅** | ⭐核心环节 |
| 15:30-16:00 | 参观新娘房/化妆间 | 检查大小、灯光、镜子 |
| 16:00-16:30 | 参观迎宾区/走廊 | 签到区、甜品台位置 |
| 16:30-17:30 | **商务洽谈** | 价格、档期、合同条款 |
| 17:30-18:30 | **试菜** | 菜单口味、摆盘、分量 |

### 酒店考察必问清单

#### 宴会厅相关

- [ ] 宴会厅具体面积是多少？层高多少？
- [ ] 是否有柱子？位置在哪里？是否影响视线？
- [ ] 最多可容纳多少桌？（50桌需要确认是否宽松）
- [ ] 是否有独立的迎宾区？面积多大？
- [ ] 舞台尺寸是多少？是否需要额外搭建？
- [ ] T台是否可以搭建？长度限制？
- [ ] 是否有LED屏幕？尺寸？是否免费使用？
- [ ] 灯光音响设备如何？是否包含在餐标内？
- [ ] 是否有专用的新娘化妆间？距离宴会厅多远？
- [ ] 同一天还有其他宴会吗？是否会影响？

#### 费用相关

- [ ] 餐标价格区间？（每桌多少钱）
- [ ] 是否有服务费？比例多少？
- [ ] 是否可以自带酒水？开瓶费多少？
- [ ] LED屏幕、灯光音响额外收费吗？
- [ ] 婚庆公司进场费多少？
- [ ] 是否有场地布置时间限制？超时收费？
- [ ] 备桌如何收费？不用是否收费？
- [ ] 定金比例？付款节点？
- [ ] 取消政策是什么？定金是否可退？
- [ ] 是否有最低消费要求？

#### 服务相关

- [ ] 婚宴当天有专门的宴会经理跟进吗？
- [ ] 每桌配备几名服务员？
- [ ] 是否提供新娘餐？
- [ ] 是否提供停车位？多少免费车位？
- [ ] 是否提供新人及父母住宿？
- [ ] 菜单可以提前多久确定？可以换菜吗？
- [ ] 是否提供请柬、席位卡等？

## Day 2（4月5日）：婚策+婚纱考察

### 上午：婚礼策划公司面谈（2-3家）

**推荐顺序**：
1. TOYOU婚礼策划
2. 幸福国度婚礼定制
3. 尤她定制婚礼

| 时间 | 事项 | 检查要点 |
|-----|------|---------|
| 09:00-10:30 | 婚策公司A面谈 | 看案例、聊方案、问价格 |
| 11:00-12:30 | 婚策公司B面谈 | 对比服务和价格 |

#### 婚策公司面谈必问清单

**公司资质**
- [ ] 公司成立多久？做过多少场婚礼？
- [ ] 是否有50桌以上大型婚礼经验？
- [ ] 是否有与我们意向酒店合作的经验？
- [ ] 是否提供一站式服务？包含哪些内容？

**案例与风格**
- [ ] 能否看真实婚礼案例照片/视频？
- [ ] 能否看我们意向酒店的实际布置案例？
- [ ] 设计图和实际效果差距大吗？
- [ ] 我们的预算和桌数，能做到什么效果？

**服务内容**
- [ ] 包含哪些人员？（策划师、设计师、督导等）
- [ ] 婚礼当天有督导现场跟进吗？
- [ ] 是否包含花艺、灯光、音响？
- [ ] 是否包含婚纱、礼服、婚车？
- [ ] 是否包含摄影摄像、司仪、化妆？

**费用相关**
- [ ] 收费标准是什么？按桌数还是按项目？
- [ ] 我们50桌的预算，能做什么档次的布置？
- [ ] 付款节点是怎样的？
- [ ] 定金多少？尾款什么时候付？
- [ ] 是否有隐形消费？
- [ ] 取消政策是什么？

**合同相关**
- [ ] 合同包含哪些具体项目？
- [ ] 效果图是否写入合同？
- [ ] 人员变动如何处理？
- [ ] 违约责任如何约定？
