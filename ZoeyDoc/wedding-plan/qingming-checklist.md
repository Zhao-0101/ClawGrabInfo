## 三、酒店考察评分表

### 评分标准（满分100分）

| 评分维度 | 权重 | 评分标准 | 泰禾洲际 | 荣誉国际 | 万达文华 | 其他 |
|----------|------|----------|----------|----------|----------|------|
| 宴会厅条件 | 25% | 面积、层高、无柱 | | | | |
| 地理位置 | 15% | 交通便利性 | | | | |
| 服务质量 | 15% | 专业度、态度 | | | | |
| 价格合理性 | 15% | 性价比 | | | | |
| 档期匹配 | 15% | 目标日期是否可订 | | | | |
| 配套设施 | 10% | 停车场、新娘房等 | | | | |
| 优惠政策 | 5% | 签约优惠力度 | | | | |
| **总分** | **100%** | | | | | |

### 详细评分项

#### 1. 宴会厅条件（25分）
- [ ] 面积是否足够（50桌需要≥500㎡）
- [ ] 层高是否满意（≥6米为佳）
- [ ] 是否无柱（无柱最佳）
- [ ] 舞台区域是否合适
- [ ] 灯光音响设备是否齐全

#### 2. 地理位置（15分）
- [ ] 距离高速路口/机场/火车站距离
- [ ] 周边交通是否便利
- [ ] 宾客导航是否容易找到
- [ ] 周边是否有住宿配套

#### 3. 服务质量（15分）
- [ ] 宴会经理专业程度
- [ ] 服务态度是否热情
- [ ] 对闽南婚俗是否了解
- [ ] 应急处理能力

#### 4. 价格合理性（15分）
- [ ] 菜单价格是否符合预算
- [ ] 服务费/酒水费是否透明
- [ ] 是否有隐藏消费
- [ ] 同档次酒店对比

#### 5. 档期匹配（15分）
- [ ] 目标婚期是否可预订
- [ ] 备选日期有哪些
- [ ] 提前多久可改期

#### 6. 配套设施（10分）
- [ ] 停车位是否充足
- [ ] 是否有新娘房/休息室
- [ ] 是否有化妆间
- [ ] 音响LED屏是否免费/收费

#### 7. 优惠政策（5分）
- [ ] 早鸟优惠
- [ ] 节假日优惠
- [ ] 赠送项目

## 四、酒店考察问题清单

### 基础信息类
1. 宴会厅的具体面积是多少？最大容纳多少桌？
2. 层高多少？有无立柱？
3. 目标婚期是否可预订？
4. 婚宴菜单有哪些档次？价格分别是多少？
5. 是否收取服务费？比例是多少？

### 费用明细类
6. LED大屏幕使用是否收费？多少钱？
7. 灯光音响是否额外收费？
8. 场地布置是否有入场费？
9. 酒水是否可以自带？开瓶费多少？
10. 取消/改期政策是什么？定金是否可退？

### 服务细节类
11. 是否提供新娘房/休息室？
12. 是否提供停车位？多少车位？
13. 是否提供签到台、桌卡等基础物品？
14. 上菜速度如何控制？
15. 是否有婚礼管家/专人对接？

### 闽南婚俗类
16. 是否熟悉泉州本地婚俗？
17. 是否可以配合敬茶改口环节？
18. 是否可以提供闽南特色菜品？

---

## 五、婚策面谈问题清单

### 公司实力类
1. 公司成立多久了？团队有多少人？
2. 一年大概做多少场婚礼？
3. 是否有50桌以上的大型婚礼经验？
4. 能否提供近期案例照片/视频？
5. 是否有固定合作酒店？

### 服务内容类
6. 套餐包含哪些服务？（布置、四大金刚等）
7. 是否包含婚礼统筹/督导服务？
8. 布置物料的材质和品质如何？
9. 是否提供婚礼前彩排？
10. 是否提供婚礼当天时间流程表？

### 价格合同类
11. 根据我们的预算，能提供什么方案？
12. 付款方式是怎样的？分几期？
13. 如果延期/取消，定金怎么处理？
14. 是否有额外收费项目？
15. 合同包含哪些条款？

### 异地婚礼特殊类
16. 对于异地婚礼，有什么特别的服务？
17. 是否可以通过线上沟通方案？
18. 婚礼前多久需要到现场确认布置？
19. 如果新人无法到场，是否可以代劳对接酒店？

## 八、紧急情况预案

| 紧急情况 | 解决方案 |
|----------|----------|
| 时间不够 | 优先考察意向最强的酒店，其他可视频考察 |
| 酒店档期冲突 | 准备2-3个备选婚期 |
| 天气不好 | 室内行程不受影响，备好雨具 |
| 身体不适 | 准备常用药品，合理安排休息 |
| 家人意见不统一 | 提前沟通，现场以新人意见为主 |
